button (icon, x, y[, ripple_x[, ripple_y[, ripple_radius[, variant[, width[, inactive[, hover[, focused]]]]]]]])
Draws a button

Parameters:

  • icon The icon to draw
  • x The position in x-axis
  • y The position in y-axis
  • ripple_x The position of the ripple in the local x-axis (optional)
  • ripple_y The position of the ripple in the local y-axis (optional)
  • ripple_radius The radius of the ripple (optional)
  • variant The type of the icon: "contained", "outlined" or "text" (optional)
  • width The width of the icon (optional)
  • inactive Shows the icon as disabled (optional)
  • hover Shows the icon as hover (optional)
  • focused Shows the outlined of the focused button (optional)
