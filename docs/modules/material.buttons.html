button (content, x, y[, ripple[, variant[, width[, elevation[, inactive[, hover[, focused]]]]]]])
Draws a button

Parameters:

  • content A string with the text or a table with the following values:
    • icon The icon of the button
    • text The text of the button
  • x The position in x-axis
  • y The position in y-axis
  • ripple A table with the following values:
    • x x ripple position in the local x-axis
    • y y ripple position in the local y-axis
    • radius radius of the ripple
    (optional)
  • variant The type of the button: "contained", "outlined" or "text" (optional)
  • width The width of the button (optional)
  • elevation The elevation of the button (optional)
  • inactive Shows the button as disabled (optional)
  • hover Shows the button as hover (optional)
  • focused Shows the outlined of the focused button (optional)
