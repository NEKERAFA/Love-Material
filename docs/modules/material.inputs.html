checkbox (x, y[, ripple[, checked[, indeterminate[, inactive[, hover[, focused]]]]]])
Draws a checkbox

Parameters:

  • x The position in x-axis
  • y The position in y-axis
  • ripple The radius of the ripple (optional)
  • checked true if is checked and false otherwise (optional)
  • indeterminate true if the checkbox is indeterminate (the checked value is omited) (optional)
  • inactive Shows the button as disabled (optional)
  • hover Shows the button as hover (optional)
  • focused Shows the outlined of the focused button (optional)
